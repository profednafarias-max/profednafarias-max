# Olá, boas-vindas ao meu espaço! 👋

Sou professora na área de tecnologia com mais de duas décadas de jornada no ensino técnico. Acredito que a programação é uma ferramenta poderosa para transformar realidades e que ninguém aprende sozinho.

### 📚 Sobre mim
- 🎓 **Atuação:** Professora de Desenvolvimento de Sistemas e Tutora Mediadora.
- 🏫 **Experiência:** 26 anos dedicados à educação, do ensino médio ao técnico e EAD.
- 💡 **Interesses:** Pedagogia inovadora, lógica de programação e desenvolvimento web.

### 🛠️ O que você encontra por aqui
Neste perfil, compartilho principalmente materiais de apoio e exercícios práticos que desenvolvo para minhas aulas:
- 🐍 Projetos simples em **Python**.
- ☕ Exemplos de lógica de programação
- 🌐 Estruturas iniciais de **HTML e CSS**.

---

### 🌱 "O aprendizado é um processo constante"
Estou sempre em busca de novas formas de ensinar e aprender. Se você é meu aluno ou aluna, sinta-se à vontade para explorar os repositórios e usar os códigos como base para seus estudos.
